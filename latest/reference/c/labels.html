Labels#

struct eqs_labels_t#

A set of labels used to carry metadata associated with a tensor map.

This is similar to a list of count named tuples, but stored as a 2D array of shape (count, size), with a set of names associated with the columns of this array (often called variables). Each row/entry in this array is unique, and they are often (but not always) sorted in lexicographic order.

Public Members

const void *labels_ptr#

internal: pointer to the rust Labels struct if any, null otherwise

const char *const *names#

Names of the variables composing this set of labels. There are size elements in this array, each being a NULL terminated UTF-8 string.

const int32_t *values#

Pointer to the first element of a 2D row-major array of 32-bit signed integer containing the values taken by the different variables in names. Each row has size elements, and there are count rows in total.

uintptr_t size#

Number of variables/size of a single entry in the set of labels

uintptr_t count#

Number entries in the set of labels
